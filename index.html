What is this?

This site checks if a word, phrase, or a sentence is a palindrome.

But what is a palindrome?

They're words, phrases, or sentences that reads the same backward as forward. For example, 'madam' and 'nurses run' are palindromes, while the word 'palindrome' itself isn't (ironic isn't it?). Palindromes also generally only count alphanumeric characters, so the sentence 'Mr. Owl ate my metal worm.' will still count as a palindrome despite not necessarily being the same backwards because of the period in the word 'Mr.'. Basically, as long as the letters or numbers match when read or written forwards and backwards, it's a palindrome!
